# To run:
* `python Proxy.py <port>` - where port is the port that the proxy should listen to. Assumes python 3.6
* Can also do `./run <port>` - Assumes python 3.6 is aliased as python3 to run (run is just a bash script running `python3 Proxy.py <port>`)
